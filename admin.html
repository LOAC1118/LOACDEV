Export & déploiement

Exportez votre configuration et préparez le déploiement.

⬇ Exporter config.json

Téléchargez le fichier de configuration complet (société, catalogue, remises, charte). À placer dans le même dossier que commande.html.

📊 Exporter catalogue CSV

Exportez uniquement le catalogue produits au format CSV pour sauvegarde ou transfert.

📄 Exporter catalogue JSON

Exportez le catalogue produits au format JSON.

📋 Ouvrir le BDC

Ouvrez le bon de commande revendeur (commande.html) dans un nouvel onglet.

📖 Guide de déploiement

1. Placez les 3 fichiers (admin.html, commande.html, config.json) dans le même dossier.

2. Configurez la société, le catalogue et les remises dans cet admin, puis cliquez Sauvegarder (le config.json est mis à jour dans le navigateur).

3. Cliquez Exporter config.json pour télécharger la configuration, remplacez le fichier dans votre dossier.

4. Partagez commande.html à vos revendeurs (par email, lien, intranet…). Ils n'ont accès qu'au bon de commande.

5. Les commandes sont reçues par email via FormSubmit à l'adresse configurée dans Société.
